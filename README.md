## ðŸŒ The Mission

In an era where **1% of companies control 90% of AI compute**, AlmaNEO proposes a new social contract. We are building an ecosystem where:
1.  **Surplus Verified Compute** flows from Big Tech to the Global South.
2.  **Ethical RLHF Data** flows from diverse human users back to AI models.
3.  **Kindness** is tokenized as a Proof of Humanity.

We leverage **Polygon's scalability** to process millions of micro-transactions, creating an "Ethical Dam" that redirects the flow of digital abundance.

### 1. GAII Dashboard (Global AI Inequality Index)
> *The Moral Lever*
A visualization tool that quantifies the **AI Divide** across 100+ countries. It serves as an ESG standard, pressuring giants to participate in our varying resource sharing initiatives.

### 2. AlmaNEO AI Hub (Strategic DePIN)
> *The Great Exchange*
Unlike traditional DePINs engaging in hardware arms races, we act as a **Resource Aggregator**.
*   **We Give**: Clean, bias-free RLHF data generated by humans in 14 language zones.
*   **We Take**: Idle GPU cloud resources from tech giants.
*   **Result**: Affordable compute for researchers and developers in the Global South.

### 3. Kindness Protocol
> *Proof of Humanity*
*   **Jeong-SBT (Soulbound Token)**: An on-chain identity that records contribution and "kindness."
*   **Gamified Contribution**: Users earn **ALMAN** tokens by providing high-quality data feedback through intuitive games.

## ðŸ’» Getting Started

This monorepo contains the full ecosystem code.

### Prerequisites
*   Node.js v20+
*   npm or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/almaneo/almaneo-org.git
    cd almaneo
    ```

2.  **Install dependencies**
    ```bash
    # Install dependencies for all workspaces
    npm install
    ```

3.  **Environment Setup**
    Create a `.env` file in the root directory based on `.env.example`.
    ```bash
    cp .env.example .env
    ```

### Running the Platform

*   **Web Interface (Landing & Dashboard)**
    ```bash
    cd web
    npm run dev
    ```
    > Runs on http://localhost:5173

*   **NFT Marketplace**
    ```bash
    cd nft
    npm run dev
    ```
    > Runs on http://localhost:5174
