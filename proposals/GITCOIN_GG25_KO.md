# Gitcoin GG25: AlmaNEO — 80억 인류를 위한 무료 AI

## 한 줄 요약

AlmaNEO는 전 세계 AI 불평등을 측정하고(GAII Index, 50개국), 누구나 무료로 AI를 사용할 수 있게 합니다(AI Hub, 14개 언어). 모든 코드는 오픈소스입니다. 670개 이상의 파일, 178K줄의 코드, 18개의 스마트 컨트랙트를 이미 배포했습니다.

---

## 우리가 만든 것 (모두 라이브)

### 1. GAII 대시보드 — AI 불평등을 숫자로 보여주다

세계 최초의 AI 불평등 복합 지수입니다. 접근성, 비용, 언어, 역량 4가지 지표로 50개국의 AI 격차를 측정합니다.

- 50개국, 4개 가중치 지표, 10개 지역
- 인터랙티브 세계지도 시각화
- PDF 리포트 다운로드 (GAII Report v1.0)
- 데이터 소스: Microsoft, World Bank, ITU, UNESCO
- 데이터 라이선스: **CC-BY-4.0** (누구나 자유롭게 사용)
- **지금 보기**: [almaneo.org/gaii](https://almaneo.org/gaii)

### 2. AI Hub — 지갑도, 신용카드도 필요 없는 무료 AI

Global South에서 AI 구독료는 월 생활비의 10~30%에 달합니다. AI Hub는 이 장벽을 없앱니다.

- **Gemini 2.5 Flash Lite** + **Llama 3.3 70B** (Groq) 두 모델 지원
- 스와힐리어, 크메르어, 인도네시아어, 베트남어, 태국어 포함 14개 언어
- 하루 50회 무료 — 결제 없음, 카드 등록 없음
- 구글/페이스북 로그인으로 바로 시작 (지갑 불필요)
- **지금 사용하기**: [almaneo.org/ai-hub](https://almaneo.org/ai-hub)

### 3. 세계문화여행 게임

글로벌 문화를 퀘스트로 배우는 교육용 게임입니다.

- 8개 지역, 20개국, 58개 이상의 퀘스트 (4종류)
- 모바일 최적화 (세로 모드, 5탭 네비게이션)
- 한국어/영어 지원
- 토큰 마이닝 연동 (8억 ALMAN 마이닝 풀)
- **지금 플레이**: [game.almaneo.org](https://game.almaneo.org)

### 4. NFT 마켓플레이스

암호화폐를 처음 접하는 사람도 가스비 걱정 없이 NFT를 만들 수 있습니다.

- ERC-721 + ERC-1155 + 렌탈(ERC-4907/5006)
- Biconomy 가스비 대납 (ERC-2771)
- POL, USDC, ALMAN 멀티 토큰 결제
- Kindness Score 기반 수수료 할인 (최대 50%)
- **지금 보기**: [nft.almaneo.org](https://nft.almaneo.org)

### 5. Kindness Protocol — 진짜 사람의 증명

온라인이 아닌 오프라인에서 만나고, 친절한 행동으로 온체인 평판을 쌓는 시스템입니다.

- 오프라인 밋업 인증 (사진 증빙)
- Ambassador SBT 4단계: Friend, Host, Ambassador, Guardian
- 밋업, 멘토링, 번역, 봉사활동 등 활동 기반 점수
- Polygon 온체인 AmbassadorSBT 컨트랙트

### 6. 스마트 컨트랙트 (Polygon Amoy 18개 배포)

모든 컨트랙트는 UUPS 업그레이드 패턴, OpenZeppelin 5.x 기반입니다.

| 컨트랙트 | 용도 |
|:---------|:-----|
| ALMANToken | ERC-20Votes 거버넌스 토큰 (80억 공급량) |
| JeongSBT | 양도 불가 영혼 토큰 (4 티어) |
| AmbassadorSBT | 밋업 기반 Kindness 평판 |
| ALMANStaking | 4단계 스테이킹 (APY 5~18%) |
| ALMANGovernor | DAO 거버넌스 (4% 쿼럼, 1주 투표) |
| ALMANTimelock | 실행 지연 (2일) |
| KindnessAirdrop | Merkle Proof 에어드롭 (6억 ALMAN) |
| TokenVesting | 팀 베스팅 (12개월 cliff + 3년) |
| MiningPool | 게임 마이닝 풀 (8억 ALMAN) |
| AlmaNFT721/1155 | NFT (렌탈 지원) |
| AlmaMarketplace | NFT 거래 (고정가/경매/렌탈) |
| AlmaPaymentManager | 멀티 토큰 결제 |
| AlmaCollectionManager | 컬렉션 관리 |

PolygonScan 검증 완료: 9개 / 18개

## 기술 스택

- **프론트엔드**: Vite 7 + React 19 + TypeScript + Tailwind CSS 3
- **백엔드**: Supabase (PostgreSQL + Auth + Storage + Realtime)
- **AI**: Google Gemini API + Groq API (Llama 3.3 70B)
- **블록체인**: Polygon (Solidity 0.8.24, OpenZeppelin 5.x, Hardhat)
- **가스비 대납**: Biconomy SDK v4 (ERC-2771)
- **인증**: Web3Auth (소셜 로그인 + 지갑)
- **호스팅**: Vercel (web, nft, game)
- **다국어**: react-i18next (14개 언어)
