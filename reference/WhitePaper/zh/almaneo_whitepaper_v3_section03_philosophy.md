## 3. 我们的理念：我们的信念

### 技术是工具，人是主体。

AlmaNEO 并非一个简单的代币项目。我们为“在人工智能时代，人类应该如何生活”这一问题提供了一个独特的答案。

--

### 3.1 情（Jeong）：从韩国走向世界

在韩国，有一个词叫做“情（Jeong）”。

情不仅仅是善良。它是一种随着时间推移而建立的深厚情谊，一种无需言语就能心意相通的心灵连接，一种即使在失去时也无法忽视的温暖。情无法用效率来解释，也无法用计算来衡量。然而，基于情的关系比任何契约都更加牢固。

人工智能追求的是优化一切：更快、更便宜、更高效。然而，情并非优化的对象。情需要时间。情有时也会显得不够高效。因此，情是人工智能无法取代的独特人类价值。

**AlmaNEO 在数字世界中体现了“正直”的价值观。**

### 3.2 三大核心原则

#### 原则一：以人为本，而非以政治为本

我们不关注国家，我们关注的是人。

一个人的价值并非取决于他的出生地或所处的政府。尼日利亚的年轻人、越南的年轻人、巴西的年轻人——他们都值得拥有同样的梦想。

AlmaNEO 不受政治意识形态或国界的束缚。我们的标准很简单：

> “这个人想要更美好的未来吗？”**

如果答案是肯定的，我们就站在他/她这边。

#### 原则二：关注个体，而非国家

庞大的体系将个体视为数字。GDP、失业率、增长率——这些统计数据掩盖了个体的生命。

但统计数据遗漏了一些东西。拉各斯一位年轻人连夜学习编程，马尼拉一位大学生兼职补贴家用，波哥大一位毕业生梦想着开办自己的小企业——他们的努力和希望无法用数字来衡量。

**AlmaNEO关注的是个体，而非统计数据。它帮助的是人，而非系统。**

#### 原则三：技术是工具，人是主体

无论人工智能发展到何种程度，最终的决策权都必须掌握在人类手中。

技术不应凌驾于人类之上，而应由人类来运用。AlmaNEO并不将人工智能神化。人工智能仅仅是一个强大的工具。如何使用它，取决于我们人类。

### 3.3 一个以善为重的世界

在当前的经济体系中，善行并不会得到回报。

你可以通过帮助他人来赚钱。你可以垄断知识而不是分享知识。你可以竞争而不是团结。善良的人往往处于劣势。

**AlmaNEO 颠覆了这一等式。**

| 分类 | 现有系统 | AlmaNEO 系统 |

| :--- | :--- | :--- |

| **行动与结果** | 垄断 → 利润 <br>共享 → 亏损 | 分享 → 提升善意值 → 更多机会 |

| **方向** | 竞争 → 成功 <br>团结 → 效率低下 | 团结 → 积累信任资产 → 在生态系统中影响力增强 <br>善意 → 链上记录 → 实际奖励 |

你帮助他人、分享知识、为社区做出贡献越多，你的价值就越高。这就是 AlmaNEO 设计的新经济模式。

### 3.4 我们梦想的未来

十年后，我们憧憬着这样一个世界：

- 一位年轻的尼日利亚开发者自由地运用人工智能工具，创办了一家致力于解决非洲农业问题的初创公司。一位越南大学生通过“善意评分”结识了一位巴西青年，并共同创业。

“善意博览会”起源于非军事区，如今已遍及世界各地贫困地区，传播希望。

“你的善意评分是多少？”这个问题，比“你的教育程度如何？”更为重要。

**这只是一个白日梦吗？**

互联网诞生之初，全球互联的世界还遥不可及。比特币诞生之初，无国界货币似乎也遥不可及。

化不可能为可能——这就是科技的力量。而引导科技朝着正确的方向发展——则是人类的职责。
