## 8. 善意博览会

### 超越线上，善意延伸至线下

区块链和人工智能是数字世界的工具。然而，AlmaNEO 所追求的“关爱”最终在人与人之间绽放。善意博览会是一个节日，来自世界各地的 AlmaNEO 社区成员齐聚一堂，共同表达团结。

![AlmaNEO 博览会](../assets/images/08.webp)

---

### 8.1 什么是善意博览会？

善意博览会是 AlmaNEO 社区的年度盛会，每年在世界各地的不同城市举办。

#### 活动宗旨

| 宗旨 | 描述 |

|------|------|

| **联结** | 以往仅在线交流的社区成员线下相聚 |

| **启发** | 分享受资助项目的成果，探索新思路 |

| **传播** |在主办地区传播 AlmaNEO 理念和人工智能素养 |

| **庆祝活动** | 庆祝并庆祝充满善意的一年 |

#### 核心项目

### 善意博览会项目

| 项目 | 详情 |

| :--- | :--- |

| **主题演讲** | AlmaNEO 年度业绩发布会、GAII 指数变化报告、分享全球人工智能不平等现状 |

| **善意奖** | 年度最善人士奖（Score Top 100 奖）、年度最具影响力项目奖、年度最佳节点运营商奖 |

| **工作坊和训练营** | 人工智能应用实践、本地青年训练营、导师-学员配对 |

| **交流互动** | 全球社区交流、合作伙伴会议、黑客马拉松 |

| **节日庆典** | 本地文化体验、社区派对、善意誓言仪式 |

### 8.2 场地选择原则

爱心博览会并非在繁华都市举办，而是在**最需要爱心的地方**举行。

#### 选择标准

| 标准 | 说明 | 权重 |

|------|------|--------|

| **人工智能可及性** | 优先考虑GAII指数较低的地区 | 30% |

| **青年人口** | 青年人口比例和失业率 | 25% |

| **基础设施** | 举办活动所需的最低基础设施 | 20% |

| **社​​区** | 活跃的当地AlmaNEO社区 | 15% |

| **安全** | 确保候选人安全 | 10% |

#### 遴选流程

1. **城市候选人征集**（DAO提案）

2. **社区讨论**（2周）

3. **DAO投票**（根据友善度评分加权投票）

4. **最终公布**（博览会前6个月）

### 8.3 资格

友善博览会面向所有人开放，但**友善人士将获得更多机会**。

| 类别 | 资格 | 申请要求 | |------|------|----------|

| **普通参与者** | ALMAN账户持有人 | 开放注册 |

| **邀请参与者** | 友善度评分前10% | 部分机票/住宿费用报销 |

| **VIP参与者** | 友善度评分前1%或获得资助者 | 全额报销 |

| **本地青年** | 主办城市居民 | 免费参与 + 训练营 |

### 8.4 首届“善意博览会”候选举办地

首届“善意博览会”的举办地点将由DAO投票决定。目前正在讨论的候选举办地：

| 候选举办地 | 特点 | 象征意义 |

|--------|------|------------|

| **非军事区（韩国）** | AlmaNEO的创始地，分裂的象征 | “超越分裂，连接彼此” |

| **内罗毕（肯尼亚）** | 非洲创业中心 | “全球南方的希望” |

| **岘港（越南）** | 快速发展的IT城市 | “机会平等” |

| **麦德林（哥伦比亚）** | 创新重生的城市 | “变革的可能” |
