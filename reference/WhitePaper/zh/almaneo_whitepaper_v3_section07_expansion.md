## 7. 生态系统拓展

### AlmaNEO 不是一个平台，而是一颗种子

AlmaNEO 的最终目标并非运营单一服务。我们的目标是创建一个平台，让世界各地的年轻人能够自主创造价值。我们设想一个良性循环：众多项目将在 AlmaNEO 生态系统中萌芽，而这些项目反过来又将丰富整个生态系统。

---

### 7.1 KINFRI：友善之友网络

#### 基于信任的全球 P2P 经济

KINFRI（友善之友）是 AlmaNEO 生态系统的首个拓展项目。它基于友善度评分，连接世界各地的年轻人，实现**人与人之间的信任交易**，而非国与国之间的贸易。

![AlmaNEO KINFRI](../assets/images/07.webp) 

### KINFRI 理念

**KINFRI 理念：**

| 参与者 |善意评分 | 角色 | |:---|:---:|:---|

| 越南青年 | 850 | 咖啡种植 → 销售 |

| 巴西青年 | 920 | 咖啡馆运营 → 咖啡豆采购 |

**KINFRI 匹配系统：**

- 基于善意评分的信任验证

- 智能合约托管

- 争议解决 DAO

**结果：** 无中介直接交易 → 公平价格 + 双方评分提升

#### KINFRI 的核心价值观

| 现有全球贸易 | KINFRI |

|-----------------|--------|

| 国与国 | 人与人 |

| 专注于大型企业 | 专注于个人和青年 | | 复杂的清关/支付 | 使用 ALMAN 代币轻松支付 |

| 信任源于品牌 | 善意评分验证信任 |

| 多重中介利润 |生产者直接对消费者交易 |

#### 可扩展领域

KINFRI 的业务范围不仅限于简单的商品交易，还涵盖以下多个领域：

| 领域 | 示例 |

|------|------|

| **农产品/食品** | 咖啡、可可和香料的直接交易 |

| **手工艺品** | 向全球销售当地工匠的作品 |

| **旅游/住宿** | 提供高友善度房东的民宿 |

| **教育/指导** | 技术指导、语言交换 |

| **自由职业服务** | 设计、翻译、开发等 |

#### 路线图

KINFRI 将在 AlmaNEO 主网稳定后，通过单独的详细规划进行开发。

| 阶段 | 时间安排 | 详情 |

| :--- | :--- | :--- |

| **第一阶段：概念验证** | 2026年下半年 | 小规模试点（咖啡直接交易） |

| **第二阶段：测试版服务** | 2027年 | 开放5个品类 |

| **第三阶段：正式上线** | 2027年下半年 | 全球扩张 |

### 7.3 AlmaNEO 大学

#### 人工智能素养，人人有责

AlmaNEO 大学是一个在线平台，提供免费的人工智能时代基础技能教育。

#### 课程设置

| 级别 | 课程 | 内容 |

|------|------|------|

| **一级** | 人工智能基础 | 什么是人工智能，如何编写提示 |

| **二级** | 人工智能应用 | 将人工智能应用于工作/学习 |

| **三级** | 人工智能开发基础 | 创建简单的人工智能应用 |

| **四级** | 人工智能伦理 | 负责任地使用人工智能 |

#### 功能

- **多语言支持**：提供主要本地语言的内容

- **社区学习**：通过导师-学员匹配进行共同学习

- **实践导向**：利用 AlmaNEO AI Hub 的真实项目

- **证书颁发**：课程完成后颁发链上证书 (SBT)

#### 教育者激励

创建高质量教育内容的贡献者将获得 ALMAN 代币奖励。

| 贡献类型 | 奖励 |

|----------|------|

| 课程内容创作 | 按内容数量 + 与学生人数成比例 |

| 翻译/本地化 | 按语言固定报酬 |

| 指导活动 | 按小时 + 友善度评分 |
