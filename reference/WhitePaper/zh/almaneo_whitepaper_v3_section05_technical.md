## 5. 技术架构

### 即使技术复杂，用户体验也应简单易用。

AlmaNEO 的技术设计原则很明确：**用户无需了解任何区块链知识。**复杂的技术在后台运行，用户以熟悉的方式使用服务。

---

### 5.1 系统概述

**AlmaNEO 系统四层架构：**

| 层级 | 组件 | 角色 |

|:---:|:---|:---|

| **1. 用户层** | AlmaNEO 应用、Web3Auth、AI 聊天界面 | 用户直接交互 |

| **2. 智能层** | AI 模型服务、DePIN 网络、模型定位 | AI 服务提供 |

| **3. 信任层** | 善意评分、生物识别认证、Jeong-SBT 发行 | 身份和贡献验证 |

| **4. 区块链层** | Polygon 网络、ALMAN 代币、智能合约 |去中心化基础设施 |

**数据流：** 用户触点 → 智能 → 信任 → 区块链（层间双向通信）

---

### 5.2 区块链：为何选择 Polygon

AlmaNEO 构建于 **Polygon 网络** 之上。

#### 选择理由

| 标准 | Polygon 的优势 |

|------|---------------|

| **Gas 费用** | 每笔交易低于 0.01 美元 — 即使对于全球南方地区的用户来说也负担得起 |

| **速度** | 2 秒内完成交易确认 — 实时交互 |

| **生态系统** | 成熟的 DeFi 和 NFT 生态系统 — 可扩展 |

| **兼容性** | 完全兼容以太坊 — 易于扩展 |

| **环境** | 基于 PoS 的节能型 — 可持续 |

#### 智能合约结构

### 智能合约结构

| 合约 |描述 | 主要作用 |

| :--- | :--- | :--- |

| **ALMAN 代币** | ERC-20 标准代币 | 总供应量：80 亿，AI 信用/质押/治理功能 |

| **Jeong-SBT** | ERC-5484 (SBT) | 不可转让的灵魂代币，链上记录善意评分 |

| **善意注册表** | 活动验证合约 | 善意活动验证与记录，对等验证投票系统 |

| **计算协议** | 资源共享合约 | DePIN 节点注册与奖励，自动计算资源分配 |

| **治理** | DAO 合约 | DAO 提案与投票，善意评分加权投票权 |

### 5.3 用户体验：零门槛设计

#### Web3Auth：5 秒启动

现有区块链服务最大的准入门槛是“创建钱包”。记下你的 12 个助记词，切勿丢失，并妥善保管你的私钥。大多数人到此为止。

**AlmaNEO 与众不同。**

![AlmaNEO 技术说明](../assets/images/05.webp)

### 传统区块链注册流程与 AlmaNEO 注册流程对比

| 类别 | 传统区块链注册流程 | AlmaNEO 注册流程 |

| :--- | :--- | :--- |

| **流程** | 安装钱包 → 生成助记词 → 安全保存 → 复制地址 → 购买代币 → 发送 → 支付 Gas 费 → 使用服务（8 个步骤） | 点击“使用 Google 登录” → 完成（2 个步骤） |

| **所需时间** | 30 分钟至 1 小时 | **5 秒** |

| **跳出率** | 90% 或更高 | 极低 |

**工作原理：**

- Web3Auth 会根据用户的社交媒体账户自动创建一个非托管钱包。

- 私钥以去中心化的方式存储，用户和 AlmaNEO 都无法访问。

- 用户无需感知钱包的存在即可使用所有功能。

#### 无 Gas 交易：无需担心手续费

区块链普及的另一个障碍是“Gas 费”。无论交易金额大小，每次交易都需要支付手续费，这对于新用户来说可能是一笔不小的负担。

**AlmaNEO 的解决方案：**

- 应用了 ERC-4337（账户抽象）技术。

- 基金会承担基本交易的 Gas 费。

- 用户可以免费使用该服务。

### 5.4 AI 基础设施：分布式且优化

#### 模型优化

AlmaNEO AI Hub 提供优化的开源 AI 模型。

| 技术 | 描述 | 效果 |

|------|------|------|

| **量化** | 模型精度调整 | 容量降低 70%，性能提升 99% |

| **LoRa** | 轻量级微调 | 本地语言优化 |

| **边缘计算** | 设备端计算 | 即使网络不稳定也能使用 |

#### DePIN 节点运行

世界各地的用户将他们的计算机连接到 AlmaNEO 网络，以提供计算资源。

如何参与节点：

1. 安装 AlmaNEO 节点软件（Windows、Mac、Linux）

2. 设置要共享的资源量（GPU、CPU、存储）

3. 连接到网络

4. 根据提供的资源量获得 ALMAN 代币奖励。

**安全性：**

- 所有计算都在 Docker 容器内的沙盒环境中运行。

- 用户数据采用端到端加密 (E2EE) 进行保护。

- 即使是节点运营者也无法查看用户查询。

### 5.6 隐私：您的数据属于您

AlmaNEO 不收集用户数据。

#### 隐私原则

| 原则 | 实施 |

|------|------|

| **无存储对话** | AI 对话不会存储在服务器上 |

| **本地加密** | 用户数据在设备上使用 AES-256 加密 |

| **匿名分析** | 数据完全匿名化，用于改进服务 |

| **零知识协议的应用** | 在验证友善度评分时保护隐私 |

> *“我们不知道您问了什么。我们只知道您有多友善。”*
