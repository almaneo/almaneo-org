### ៩.៣ យន្តការបោះឆ្នោត

#### ការបោះឆ្នោតដែលមានទម្ងន់ដោយសប្បុរស

```
អំណាចបោះឆ្នោត = √(សញ្ញាសម្ងាត់ដែលកាន់កាប់) × គុណពិន្ទុសប្បុរស
```

| ពិន្ទុសប្បុរស | គុណ |
|----------------|------|
| ០ - ១០០ | ១.០x |
| ១០១ - ៥០០ | ១.៥x |
| ៥០១ - ១០០០ | ២.០x |
| ១០០១+ | ២.៥x |

- ពង្រីកសំឡេងរបស់អ្នករួមចំណែកសកម្មក្នុងសហគមន៍
- រចនាសម្ព័ន្ធមួយដែលសេចក្តីសប្បុរសស្មើនឹងឥទ្ធិពល
